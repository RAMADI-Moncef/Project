Git et GitHub - Notions Fondamentales

Git est un système de contrôle de version distribué utilisé pour suivre les modifications apportées au code source au cours du développement logiciel. GitHub est une plateforme de collaboration qui permet aux développeurs d'héberger et de gérer leurs projets Git.

Commandes de Base Git

Initialiser un dépôt Git

git init

Cloner un dépôt

git clone https://github.com/nom-utilisateur/nom-depot.git

Ajouter des fichiers au suivi

git add nom-fichier

Committer des modifications

git commit -m "Message de commit"

Pousser les modifications vers un dépôt distant

git push origin main

Utilisation de GitHub

GitHub permet de collaborer facilement avec d'autres développeurs, de gérer des pull requests et de suivre les problèmes. Voici comment créer une nouvelle branche et faire une pull request :

Créer une nouvelle branche

git checkout -b nom-branche

Faire une pull request

Après avoir poussé votre branche sur GitHub, allez sur la page du dépôt et cliquez sur "New pull request".
